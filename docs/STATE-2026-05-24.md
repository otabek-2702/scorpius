## 4. Architecture state — high level

**Frontend:** Next.js 16 App Router · React 19 · Tailwind v4 · Newsreader serif + Geist sans
+ JetBrains Mono. Light theme (cream + ink + gold). Apple-mode craft pass applied.

**Card DSL:** 12 typed lesson card variants in `lib/lesson.ts` — `intro`, `discover`,
`explainer`, `mcq`, `numberline`, `sequence`, `sort`, `diagram`, `story`, `ask`, `done`,
plus the special Brachistochrone simulation card. Each renders to a typed React component
in `components/learn/cards.tsx`. The deck (`components/learn/LessonDeck.tsx`) does
vertical-snap scroll with answer-lock gating.

**AI adapter:** `lib/ai/model.ts` exposes one `LanguageModel` interface. Swap providers by
which env key is set (`OPENAI_API_KEY` → OpenAI, `GEMINI_API_KEY` → Gemini). Production runs
OpenAI gpt-5-mini for text + gpt-image-1 for images + gpt-5.1 vision for offline curriculum
extraction. Gemini is wired but billing-blocked in UZ.

**Backend:** Firebase Firestore + firebase-admin 13. Server writes via `lib/firebase/admin.ts`
(lazy Proxy init, BOM/CRLF sanitizer). Client reads via `lib/firebase/client.ts`. Three
collections actively used: `waitlist/`, `images/` (SHA cache), `students/{id}/emaktab/snapshot`
(seeded from the Playwright scraper).

**Caching:** every expensive AI call hashes its input (`sha256(prompt+size+quality)`) and
caches the output. First student pays, every subsequent student is free.

**Build/deploy:** Vercel. `vercel.json` declares `framework: nextjs` explicitly (autodetect
was failing and serving only `/public`). `lib/firebase/admin.ts` uses a lazy Proxy so
`next build` doesn't crash on missing env vars at build time.
